# DevPace

個人のPC作業活動をバックグラウンドで記録し、開発の「密度」を可視化するためのRust製CLIツール。

## 特徴
- **自動記録**: PCログイン時に自動的にバックグラウンドで起動し、アクティブウィンドウと操作回数(Ops)を記録します。
- **DeepThinkingの判定**: 離席(Away)と、思考中(DeepThinking)を自動判別し、ただの放置ではない「没頭している時間」を可視化します。
- **軽量・高速**: Rustで記述されており、リソース消費を最小限に抑えています。
